# CLI_utilities

Useful scripts and aliases at your fingertips.

Be warned, they may burst your productivity!

### Installation

```
sudo cp nameofthescript.sh /usr/local/bin/nameofthescript
sudo chmod +x /usr/local/bin/nameofthescript
```
